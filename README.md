# 24hourlogo
my portfolio of logos made in 24 hours or less and request

## GitHub Pages

This repo deploys to GitHub Pages using GitHub Actions from the `24-hour-logos` folder.

1. Go to **Settings → Pages**.
2. Set **Source** to **GitHub Actions**.
3. Push to `main` to trigger `.github/workflows/static.yml`.
4. After the workflow passes, visit: `https://lumpyspaceprogrammer.github.io/24hourlogo/`
